# Vendored Arduino libraries

## SparkFun_BNO08x_Arduino_Library

Upstream: https://github.com/sparkfun/SparkFun_BNO08x_Arduino_Library

Used for optional BNO086 terrain compensation (`COMPILE_TERRAIN_COMP` in `Firmware/RTK_Surveyor/RTK_Surveyor.ino`).

Install via Arduino Library Manager as **SparkFun BNO08x Cortex Based IMU**, or add this folder to your library path.
